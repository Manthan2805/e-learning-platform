The animation-duration property in CSS is used to define how long an animation should take to complete one cycle.

property values:

  • time: Specifies animation duration in seconds/milliseconds.
  • initial: This value is used to set the property to its default value.
  • inherit: This value is used to inherit the property from its parent element.

For example, animation-duration: 5s; will set duration of animation
