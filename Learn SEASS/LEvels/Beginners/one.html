Welcome to Learn Seass, a game where you can code and learn styling your website using CSS! Let's start with colors, who doesn't need them in website? Use background property, which changes the color of the background and accepts the following values:

  • background-color: Specifies the background color to be used.
  • background-image: Specifies ONE or MORE background images to be used.
  • background-position: Specifies the position of the background images.
  • background-repeat: Specifies the size of the background images.

For example, background-color: pink; will change the background color to pink
